About Me

I am a second-year Computer Science Ph.D. student at Portland State University, with a deep passion for machine learning and its real-world applications. My current research focuses on Large Language Models (LLMs) and Retrieval-Augmented Generation (RAG), and I am driven by the goal of becoming a Machine Learning Engineer who pushes the boundaries of how AI can solve practical problems.

While my academic background is rooted in research, I possess strong programming skills and am actively seeking to apply them in more application-oriented development. This portfolio is a showcase of my journey in bridging advanced machine learning concepts with hands-on software engineering.

My Projects

Titanic Survival Prediction – A Full-Stack ML System

An end-to-end machine learning project to predict Titanic passenger survival. I independently developed a custom neural network with PyTorch, created an ensemble model with Scikit-learn and XGBoost, and built a Flask RESTful API for real-time predictions. The system is containerized with Docker and features a responsive frontend, showcasing my skills in both machine learning and full-stack development.

Tech Stack: PyTorch, Scikit-learn, XGBoost, Flask, Docker, Pandas, HTML/CSS, JavaScript
GitHub

Speed Grader Pro – Desktop Grading Assistant

A practical desktop application built with Python and Tkinter to streamline the grading process. The app features automated score calculation, bonus point handling, and a user-friendly GUI with input validation. It was packaged into a standalone Windows executable using PyInstaller, demonstrating my ability to create useful tools for specific needs.

Tech Stack: Python, Tkinter, PyInstaller, GUI Design
